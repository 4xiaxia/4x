# AIClient-2-API 部署指南

## 🎯 核心信息

**无需 Docker 也能部署！** 本项目支持两种部署方式，本地直接运行不需要 Docker。

---

## 🚀 快速开始（3 步）

```bash
# 1. 进入项目目录
cd /workspace/AIClient-2-API

# 2. 一键部署
bash 一键部署.sh

# 3. 访问服务
# 浏览器: http://localhost:3000
# 密码: admin123
```

就这么简单！

---

## 📋 部署方式对比

| 方式 | 需要 Docker | 启动时间 | 推荐度 |
|------|-----------|---------|--------|
| **本地直接运行** | ❌ 不需要 | ~10秒 | ⭐⭐⭐⭐⭐ |
| **Docker 容器** | ✅ 需要 | ~30秒 | ⭐⭐⭐ |

---

## 🔍 环境检测

不确定你的环境支持哪种方式？运行检测脚本：

```bash
bash check-environment.sh
```

会自动检测并推荐最适合的部署方案！
