## 📱 傻瓜版界面功能

### 首页功能

#### 🎯 连接状态检测
- 自动检测服务是否正常运行
- 实时显示状态（正常/未启动）

#### 🌟 推荐服务（免费）
1. **Gemini（Google）**
   - 最简单，推荐新手
   - 用 Google 账号登录
   - 授权流程最快

2. **通义千问（阿里）**
   - 国内友好
   - 用手机号登录
   - 速度快

#### ⚡ 其他服务
1. **Kiro（Claude）**
   - 需要先安装 Kiro 客户端
   - 质量最高

2. **OpenAI**
   - 需要自己的 API Key
   - 适合已有账号的用户

### 授权页面功能

#### 🔐 一键授权
- 点击按钮自动打开浏览器
- 显示清晰的授权步骤
- 实时检查授权状态
- 自动轮询等待完成

#### 📝 步骤提示
1. 点击按钮，浏览器自动打开
2. 在浏览器中登录账号
3. 点击"允许"完成授权
4. 回来查看授权成功

### API Key 输入功能

#### 🔑 OpenAI API Key
- 安全的密码输入框
- 获取 API Key 的链接
- 一键保存配置

## 🆘 常见问题

### Q1: 授权失败怎么办？
**A**: 重新点击"一键授权"按钮，确保能访问对应的服务（Gemini 需要能访问 Google）。

### Q2: 看到"服务未启动"？
**A**: 运行 `bash 快速开始.sh` 启动服务。

### Q3: 想用多个 AI？
**A**: 可以！傻瓜版支持配置多个 AI，在首页点击不同的卡片即可。

### Q4: 如何切换到高级版？
**A**: 在傻瓜版底部点击"⚙️ 高级模式"按钮，或者直接访问 http://localhost:3000

### Q5: 需要懂技术吗？
**A**: 完全不需要！傻瓜版就是为小白设计的，只需要会用鼠标点击就行。
