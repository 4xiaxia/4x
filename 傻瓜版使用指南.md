# 🎉 傻瓜版使用指南

## 不懂技术？完全没问题！

---

## 🚀 3 步开始使用

### 第 1 步：启动服务（1 分钟）

打开终端，运行：

```bash
cd /workspace/AIClient-2-API
bash 一键部署.sh
```

看到 ✅ 就成功了！

---

### 第 2 步：打开傻瓜版界面（5 秒）

浏览器打开：**http://localhost:3000/simple.html**

这就是傻瓜版！比普通版简单 100 倍！

---

### 第 3 步：一键授权（3 分钟）

#### 方式 A：用 Gemini（推荐，最简单！）

1. 点击卡片 **"🌟 Gemini（Google）"**
2. 点击按钮 **"🔐 一键授权"**
3. 浏览器自动打开 Google 登录页面
4. 用你的 Google 账号登录
5. 点击"允许"
6. 回到这个页面，看到 **"✅ 授权成功！"**

**搞定！**现在可以用 Gemini 了！

#### 方式 B：用通义千问

1. 点击卡片 **"🚀 通义千问（阿里）"**
2. 点击按钮 **"🔐 一键授权"**
3. 浏览器打开阿里云登录页面
4. 用手机号登录
5. 授权成功！

#### 方式 C：用 OpenAI（需要 API Key）

1. 点击卡片 **"🔑 OpenAI"**
2. 粘贴你的 API Key（从 OpenAI 官网获取）
3. 点击"保存"

---

## 🎯 完成后怎么用？

### 在软件中配置 AI

任意支持 OpenAI 的软件都可以用！

#### Cherry-Studio 配置：
- **API 地址**：`http://localhost:3000/v1`
- **API 密钥**：随便填（比如 `sk-test`）
- **模型名称**：`gemini-2.5-flash-exp`（用 Gemini）或 `qwen3-coder-plus`（用千问）

#### NextChat 配置：
- **Base URL**：`http://localhost:3000/v1`
- **API Key**：随便填
- **模型**：同上

#### 其他软件（Cline、Cursor 等）：
都是类似的配置，找到"API 地址"和"API Key"填入即可！

## 🤔 有问题？

### 问题 1：授权失败怎么办？

**Gemini 授权失败：**
- ✅ 确保能访问 Google（如果需要翻墙）
- ✅ 清除浏览器缓存，用无痕模式试试
- ✅ 检查端口 8085 是否被占用

**千问授权失败：**
- ✅ 确保能访问阿里云服务
- ✅ 检查手机号是否正确

### 问题 2：看到"服务未启动"？

运行这个命令：
```bash
cd /workspace/AIClient-2-API
bash 一键部署.sh
```

等待看到 ✅ 就行了！

### 问题 3：想换别的 AI？

直接在傻瓜版界面点击其他卡片，重新授权就行！
