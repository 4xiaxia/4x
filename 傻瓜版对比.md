## 🚀 使用流程对比

### 傻瓜版流程（3 步）

```
1. 选择 AI（点击卡片）
   ↓
2. 点击"一键授权"
   ↓
3. 浏览器登录

✅ 完成！
```

### 高级版流程（10+ 步）

```
1. 打开"提供商池"页面
   ↓
2. 找到要配置的提供商
   ↓
3. 点击"生成授权"
   ↓
4. 选择授权方式
   ↓
5. 浏览器登录
   ↓
6. 回到页面检查状态
   ↓
7. 等待凭证文件生成
   ↓
8. 打开"配置文件"页面
   ↓
9. 找到刚生成的凭证
   ↓
10. 手动关联到提供商
   ↓
11. 保存配置
   ↓
12. 检查健康状态

✅ 完成（如果中间没出错）
```

## 🆘 问题处理对比

### 傻瓜版遇到问题：

```
问题：授权失败
解决：重新点击"一键授权"
耗时：10 秒
```

### 高级版遇到问题：

```
问题：Token 过期
步骤：
1. 查看"实时日志"定位错误
2. 分析日志信息
3. 确定是 Token 过期
4. 找到对应提供商
5. 点击"刷新 Token"
6. 等待刷新完成
7. 检查凭证文件
8. 验证健康状态
耗时：5-10 分钟
```
