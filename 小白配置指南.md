## 第 1 步：启动服务（1 分钟）

打开终端，进入项目目录，运行：

```bash
cd /workspace/AIClient-2-API
bash 一键部署.sh
```

看到 ✅ 就说明成功了！

---

## 第 2 步：打开网页（10 秒）

浏览器打开：**http://localhost:3000**

输入密码：**admin123**

登录成功！

---

## 第 3 步：授权一个 AI 账号（3 分钟）

### 方式 A：使用 Gemini（推荐，最简单）

1. 点击左侧菜单 **"提供商池"**
2. 找到 **"Gemini CLI OAuth"**，点击右上角 **"生成授权"**
3. 浏览器会自动打开 Google 登录页面
4. 登录你的 Google 账号
5. 点击"允许"
6. 等待几秒，授权成功！

✅ 现在你可以用 Gemini 了！

### 方式 B：使用 Qwen（通义千问）

1. 点击左侧菜单 **"提供商池"**
2. 找到 **"Qwen CLI OAuth"**，点击 **"生成授权"**
3. 浏览器打开阿里巴巴登录页面
4. 使用手机号登录
5. 授权成功！

## ❓ 常见问题

### Q1：授权失败怎么办？

**Gemini 授权失败：**
- 确保网络能访问 Google
- 清除浏览器缓存，用无痕模式试试
- 检查端口 8085 是否被占用（用命令 `lsof -i :8085`）

**Qwen 授权失败：**
- 确保能访问阿里云服务
- 检查手机号是否正确

### Q2：想用其他 AI 怎么办？

**想用 Claude：**
- 需要先安装 Kiro 客户端
- 登录 Kiro 后，凭证文件会自动生成
- 在网页中上传凭证文件

**想用 OpenAI：**
- 需要自己有 OpenAI API Key
- 在网页的"配置"页面填入 API Key

### Q3：配置太多了看不懂？

**别看那么多！只需要：**

1. **启动服务**：`bash 一键部署.sh`
2. **打开网页**：http://localhost:3000
3. **授权账号**：点"生成授权"就行

其他的配置默认就好，不用改！

### Q4：忘记密码了怎么办？

编辑文件：`configs/pwd`

把里面的内容改成新密码，保存后刷新网页即可。

### Q5：服务启动失败？

```bash
# 查看日志
tail -f /tmp/aiclient.log

# 检查端口是否被占用
lsof -i :3000

# 如果被占用，杀掉进程
kill -9 <进程ID>
```

## 🎯 核心概念（一分钟看懂）

### 这个项目是干什么的？

**简单说：把免费的 AI 账号，变成标准的 API**

比如：
- 你有 Gemini 免费账号（只能在网页用）
- 通过这个项目，就能在其他软件里用 Gemini

### 工作流程

```
你的软件 → AIClient-2-API → Google/Gemini → 返回结果
         (转换协议)       (免费账号)
```

### 为什么需要这个？

1. **省钱**：用免费账号，不用付 API 费
2. **统一**：不同 AI 用同一个接口
3. **灵活**：可以随时切换 AI
